UTM Housing Platform UX Project

This project focused on improving the rental booking cancellation flow for a student housing platform designed for UTM students. The work combined interface design, medium-fidelity prototyping, and think-aloud usability testing.

Project Overview

The usability task asked users to locate an existing rental booking and cancel it using the platform without external help. The goal was to test whether the workflow was clear, usable, and easy to complete.

Problem

Users were able to complete the cancellation task, but several parts of the interface created friction. Participants had trouble finding the booking management area, understanding labels, noticing important buttons, and confirming refund or cancellation status.

My Role

I worked in a team for this project, and it has helped me improve my teamwork and collaborative skills in the context of the design process. My main responsibility in the project was to design the medium-fidelity prototype, and in this part, I tried to focus on the interface, user flow, and visual clarity. However, apart from the design part, I also contributed to the team in other ways, like helping to recruit participants, usability feedback, and arranging the data from the interview sessions. I also contributed to the analysis of the feedback from the participants and the writing of the reports.

What I Learned

This project showed me that a task can be functional while still creating uncertainty. I learned that label clarity, visual emphasis, and next-step communication are critical in transactional UX.

3D Modeling for Kitty’s Wonderland

This project shows my 3D modeling process in Autodesk Maya for environmental assets designed for Kitty’s Wonderland, a cozy first-person cat exploration game set in a bedroom.

Project Overview

I created furniture models including a bed frame and wardrobe to support the game’s warm, domestic environment. The goal was to build clear, readable assets that fit the relaxed, cat-scale world of the game.

Process

I started with simple blockout forms, then shaped the furniture structure step by step in Maya. I refined the silhouette, adjusted proportions, developed the main forms, and then moved into UV mapping and material application.

What I Learned

This project helped me improve my workflow in 3D asset development, especially in modeling, UV unwrapping, material setup, and rendering. I also learned how individual assets support a larger worldbuilding concept.

Graphic Design Poster: Astroworld Tour

This project is a promotional poster and public-space mock-up created for a graphic design and marketing assignment. I designed the poster around Travis Scott’s Astroworld Tour using space-inspired imagery, contrast, and layered typography.

Project Overview

I created a concert poster and then composited it into a billboard-style street display. The design uses a diagonal title, grouped tour dates, sky background, and floating imagery to build a surreal, large-scale atmosphere.

Typography and Layout

I used Ebrima for readable information and a stronger display typeface for emphasis. This created a clearer visual hierarchy while keeping the poster energetic and expressive.

What I Learned

This project improved my understanding of composition, visual hierarchy, contrast, and poster mock-up design. I learned how layout, typography, and color can communicate both event information and mood.

Fan Page Website Design

This project is a fan page website designed and coded in HTML, CSS, and JavaScript. It presents NF’s music and background through a dark visual theme, album artwork, structured content sections, and interactive buttons.

Project Overview

I designed this site as a themed fan page with a strong visual identity. The page includes a top banner, featured album image, artist biography, and interactive elements that encourage visitors to engage with the content.

Interaction Design

The site includes both BOM and DOM event handling. On page load, a welcome alert appears. One button asks the user to confirm before opening NF’s YouTube page in a new tab, while another button updates the fan message text directly on the page.

What I Learned

This project helped me practice building a themed front-end experience with structure, styling, and simple interaction. I learned how visual tone, content organization, and JavaScript events work together to shape how users experience a page.
